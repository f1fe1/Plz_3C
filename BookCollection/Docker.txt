
Docker - це платформа для розробки, доставки та запуску застосунків у контейнерах. 
Контейнери - це виконувані одиниці програмного забезпечення, які включають у себе код, 
середовище виконання, бібліотеки та інші необхідні ресурси. Docker надає можливість упаковувати програмні 
застосунки та їх залежності в контейнери, які можуть бути відтворені на будь-якій системі, що підтримує Docker.

включає:
1.Контейнери: Docker використовує контейнери для упаковки та виконання програм. Кожен контейнер включає в себе 
все необхідне для виконання програми, включаючи код, середовище виконання, бібліотеки та інші залежності.
2.Dockerfile: Dockerfile - це текстовий файл, що містить інструкції для автоматизованого створення образу Docker.
3.Образи Docker: Образ Docker - це запакований, готовий до використання шаблон для створення контейнера.
4.Docker Hub: Docker Hub - це публічний реєстр образів Docker, де розробники можуть зберігати та ділитися своїми образами. 
5.Docker Compose: Docker Compose - це інструмент для визначення та запуску багатоконтейнерних застосунків. 
6.Docker Swarm та Kubernetes: Docker Swarm і Kubernetes - це оркестратори контейнерів, які дозволяють керувати та масштабувати багатоконтейнерні застосунки.
